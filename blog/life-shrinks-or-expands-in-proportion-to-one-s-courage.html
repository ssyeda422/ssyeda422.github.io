Customizable Browser Homepage

Saniya Syeda
16 Oct 2020

This application was developed as part of a final project for a webware course with three other group members. Link to Github Repo

Description: A browser home page application that has customizable boxes/categories that users can fill in to have bookmarks, notes, tasks, and images. The idea is that the user will open their browser and use this page to start their day/work/session. The user will be able to log in so that their content persists between sessions. This application is targeted at users who use their computers on a daily basis and would like a customizable web browsing experience. The application will make use of a database in MongoDB to store the homepage layout, input, preferences, etc.
